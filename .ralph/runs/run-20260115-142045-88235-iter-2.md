## Implementation Details

### Button Component
- Location: `Sources/Voxa/MainWindow.swift` → `HomeContentView.startRecordingButton`
- Shape: `Capsule()` for pill shape
- Background: `Color.Voxa.accent` (terracotta) → `Color.Voxa.error` (red) when recording

### Pulse Animation
- State: `isPulsing` controls animation
- Effect: Circle with `.repeatForever` animation that scales 1.0→1.4 and fades
- Trigger: Starts when `isRecording` becomes true

### Keyboard Shortcut Badge
- Component: `shortcutBadge` computed property
- Display: "⌘⇧Space" in `Font.Voxa.monoSmall`
- Style: Semi-transparent white capsule background (0.15 opacity)

### Hover Lift Effect
- State: `isRecordingButtonHovered` tracks hover
- Effects on hover:
  - Shadow radius: 6 → 12
  - Shadow Y offset: 2 → 4
  - Scale: 1.0 → 1.02
  - Y offset: 0 → -2 (lift)

### Recording Functionality
- Toggle action posts `.toggleRecording` notification
- AppDelegate observes `.toggleRecording` and calls `toggleRecordingFromHotkey()`
- AppDelegate posts `.recordingStateChanged` with `RecordingState` object
- Button subscribes to `.recordingStateChanged` to update `isRecording` state
